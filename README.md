# 🌍 ExploreMundo - Site de Viagens

## 📌 Sobre o Projeto

O **ExploreMundo** é uma página web desenvolvida em HTML e CSS com o objetivo de apresentar destinos turísticos de forma visual e atrativa. O site simula uma plataforma de viagens, destacando locais populares, informações institucionais e um formulário de contato.

---

## 🎯 Objetivo

Criar uma interface moderna e responsiva para exibir:

* Destinos turísticos
* Informações sobre a empresa
* Formulário de contato
* Localização e dados de contato

---

## 🛠️ Tecnologias Utilizadas

* **HTML5** → Estrutura da página
* **CSS3** → Estilização e layout
* **Flexbox** → Organização dos elementos
* **Media Queries** → Responsividade

---

## 🖥️ Estrutura do Site

### 🔹 Navbar (Menu Superior)

* Nome da marca: **ExploreMundo**
* Links de navegação:

  * Início
  * Destinos
  * Sobre
  * Contato

---

### 🔹 Banner Principal

* Imagem de destaque (Bangkok)
* Texto chamativo:

  * "Descubra Lugares Incríveis!"
* Botão: **Saiba Mais**

---

### 🔹 Destinos Populares

Seção com cards contendo:

* Imagens
* Títulos
* Descrição

Exemplos:

* Praias paradisíacas
* Montanhas
* Cidades históricas

Botão:

* **Ver Mais Destinos**

---

### 🔹 Sobre Nós

* Pequena descrição institucional
* Botão: **Leia Mais**

---

### 🔹 Contato

Dividido em duas partes:

#### 📩 Formulário

* Nome
* E-mail
* Mensagem
* Botão de envio

#### 📍 Localização

* Imagem de mapa
* E-mail
* Telefone
